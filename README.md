# EventHub — Event and Participant Management System

Full-stack web application built as part of the Web Programming 2026 course.

## Project Structure

- `backend-django/` — Django REST API (main backend)
- `backend-node/` — Node.js/Express API (comparative backend)
- `frontend/` — React SPA
- `docs/` — Technical report and presentation

## Tech Stack

- **Backend 1** : Python / Django / Django REST Framework
- **Backend 2** : Node.js / Express
- **Frontend** : React (Vite)
- **Database** : PostgreSQL
- **Auth** : JWT (SimpleJWT)
